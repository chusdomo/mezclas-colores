🔴 ROJO con otros colores

Guía completa de mezclas y combinaciones cromáticas

🔸 Rojo + Azul = Violeta

🔴 Rojo = 100%
🔵 Azul = 100%
➡ Violeta = 50% + 50%
🔸Más Rojo (70/30): violeta cálido, más cercano al magenta
🟣 Ideal para flores brillantes, telas mágicas, detalles llamativos
🔸Más Azul (30/70): violeta profundo, más frío
🔵 Ideal para sombras, capas de profundidad, criaturas nocturnas
🌌Usos: Magia, efectos mágicos, telas de fantasía, zonas sombrías
⚠Consejo: Mezclar bien para evitar vetas. Añadir blanco crea tonos lavanda útiles

🔸 Rojo + Amarillo = Naranja

🔴 Rojo = 100%
🟡 Amarillo = 100%
➡ Naranja = 50% Rojo + 50% Amarillo
🔸Más Rojo (70/30): rojo-naranja intenso
🟥 Ideal para ladrillos, óxido reciente, zonas calcinadas
🔸Más Amarillo (30/70): naranja dorado
🟨 Ideal para luces cálidas, atardeceres, campos secos
🔥Usos: Llamas, ropas cálidas, faroles, óxido
✅Consejo: Añadir blanco da melocotón. Añadir gris = naranja tostado

🔸 Rojo + Verde = Marrón

🔴 Rojo = 100%
💚 Verde = Amarillo 60% + Azul 40%
➡ Marrón terroso = 50% Rojo + 50% Verde
🔸Más Rojo (70/30): marrón cálido rojizo
Ideal para arcillas, maderas rojizas, óxido seco
🔸Más Verde (30/70): marrón oliva
Perfecto para vegetación seca, suelos húmedos
🌳Usos: Tierra, moho, bases de vegetación, madera natural
👀Consejo: El rojo neutraliza el verde y da calidez. Combinar con blanco = beige tierra

🔸 Rojo + Naranja = Rojo-naranja

🔴 Rojo = 100%
🟧 Naranja = Rojo 50% + Amarillo 50%
➡ Rojo-naranja cálido = 50% Rojo + 50% Naranja
🔸Más Rojo (70/30): rojo encendido, vibrante
Ideal para faroles, fuego intenso, detalles mágicos
🔸Más Naranja (30/70): naranja rojizo
Usos: telas cálidas, calabazas, detalles de estructuras
🔥Usos: Magia, fuego, tela ceremonial, decoraciones festivas
🎨Consejo: Muy saturado, combinar con gris o azul para enfriar y equilibrar

🔸 Rojo + Violeta = Rojo-violeta

🔴 Rojo = 100%
🟣 Violeta = Rojo 50% + Azul 50%
➡ Rojo-violeta = 50% Rojo + 50% Violeta
🔸Más Rojo (70/30): tono vino claro, cálido
🍷 Ideal para telas de nobleza, estandartes
🔸Más Violeta (30/70): púrpura más oscuro
🎭 Ideal para sombras de piel, oscuridad con tono cálido
🎨Usos: Ropas reales, sombras cálidas, luces crepusculares
✅Consejo: Agregar blanco crea tonos rosados místicos

🔸 Rojo + Azul-verde = Grisáceo oscuro

🔴 Rojo = 100%
🌊 Azul-verde = Azul 60% + Verde 40%
➡ Gris oscuro con matiz violáceo
🔸Más Rojo (70/30): gris cálido violáceo
Ideal para sombras cálidas, paredes desgastadas
🔸Más Azul-verde (30/70): gris frío con matiz verdoso
Perfecto para rocas, estructuras húmedas
🏚Usos: Sombras, metales oxidados, madera antigua
⚠Consejo: Muy útil para neutralizar zonas demasiado saturadas

🔸 Rojo + Amarillo-verde = Marrón cálido

🔴 Rojo = 100%
💚 Amarillo-verde = Amarillo 70% + Azul 30%
➡ Marrón con matiz oliva cálido
🔸Más Rojo (70/30): marrón anaranjado, seco
Ideal para desiertos, cuero seco
🔸Más Amarillo-verde (30/70): marrón oliva
Perfecto para follaje seco, corteza
🌾Usos: Vegetación seca, barro claro, tierras soleadas
✅Consejo: Añadir blanco → beige natural. Buen sustituto del siena tostado

🔸 Rojo + Amarillo-naranja = Naranja intenso

🔴 Rojo = 100%
🍊 Amarillo-naranja = Amarillo 60% + Rojo 40%
➡ Naranja encendido, muy saturado
🔸Más Rojo (70/30): rojo-anaranjado brillante
Ideal para efectos de fuego, faroles
🔸Más Amarillo-naranja (30/70): naranja cálido
Perfecto para piel de personajes, tonos cálidos
🔥Usos: Llamas, efectos mágicos, criaturas ígneas
⚠Consejo: Muy potente. Suavizar con blanco o gris si distrae

🔸 Rojo + Rojo-naranja = Rojo quemado

🔴 Rojo = 100%
🟥 Rojo-naranja = Rojo 60% + Amarillo 40%
➡ Rojo oscuro con matiz cálido, como teja
🔸Más Rojo (70/30): rojo vino con un toque anaranjado
Ideal para óxido viejo, cuero desgastado
🔸Más Rojo-naranja (30/70): rojo ladrillo
Perfecto para casas rurales, muros antiguos
🏚Usos: Tejados, muros, óxido seco, detalles envejecidos
🎨Consejo: Agregar gris lo convierte en rojo inglés, muy versátil

🔸 Rojo + Azul-violeta = Púrpura oscuro

🔴 Rojo = 100%
🔵🟣 Azul-violeta = Azul 60% + Rojo 40%
➡ Púrpura denso, muy profundo
🔸Más Rojo (70/30): borgoña con base azulada
Ideal para detalles lujosos, vino, telas caras
🔸Más Azul-violeta (30/70): púrpura frío y oscuro
Perfecto para sombras profundas, capuchas mágicas
🍷Usos: Tela mágica, vino, oscuridad rica, profundidad visual
✅Consejo: Añadir blanco da tonos malva muy interesantes

🔸 Rojo + Rojo-violeta = Burdeos

🔴 Rojo = 100%
🍷 Rojo-violeta = Rojo 60% + Azul 40%
➡ Burdeos elegante y oscuro
🔸Más Rojo (70/30): burdeos cálido
Perfecto para capas, trajes ceremoniales
🔸Más Rojo-violeta (30/70): más púrpura, más sobrio
Usos: detalles mágicos, sombras realistas
🎩Usos: Ropas de nobleza, pieles oscuras, madera barnizada
👑Consejo: Mezcla estable, ideal para bases rojizas y profundidad

🔸 Rojo + Blanco = Rosa

🔴 Rojo = 100%
⚪ Blanco = 100%
➡ Rosa claro, pastel o saturado según cantidad
🔸Más Rojo (70/30): rosa saturado
Perfecto para flores, detalles mágicos, carnes vivas
🔸Más Blanco (30/70): rosa pálido
Ideal para piel clara, nubes, ropas delicadas
🌸Usos: Flores, telas, rubor, zonas de luz
✅Consejo: Añadir azul = lila. Con negro → rosa sucio muy útil

🔸 Rojo + Negro = Rojo profundo

🔴 Rojo = 100%
⚫ Negro = 100%
➡ Rojo oscuro, tipo sangre seca
🔸Más Rojo (70/30): rojo sangre
Perfecto para heridas, sangre fresca, telas densas
🔸Más Negro (30/70): rojo casi marrón
Ideal para sombras cálidas, óxido profundo
🩸Usos: Sangre, heridas, capas, armaduras oscuras
⚠Consejo: Añadir azul = púrpura profundo. Controlar para no ennegrecer

🔸 Rojo + Gris = Rojo apagado

🔴 Rojo = 100%
⚪ Gris = Negro + Blanco
➡ Rojo mate, de baja saturación
🔸Más Rojo (70/30): rojo algo desaturado
Perfecto para ladrillos, telas viejas
🔸Más Gris (30/70): rojo casi marrón grisáceo
Ideal para tierra húmeda, uniformes, desgaste
🏗Usos: Muros, suelo, desgaste de estructuras, vestimenta realista
🎯Consejo: Muy útil para neutralizar zonas demasiado brillantes
