💚 VERDE con otros colores

Guía completa de mezclas y combinaciones cromáticas

🔸 Verde + Rojo = Marrón

💚 Verde = Amarillo 60% + Azul 40%
🔴 Rojo = 100% Rojo
➡ Marrón neutro con tono terroso
🔸Más Verde (70/30): marrón oliva
Ideal para bases vegetales, árboles
🔸Más Rojo (30/70): marrón rojizo
Perfecto para óxido, arcilla seca
🌳Usos: Vegetación seca, madera, bases de terreno realista
✅Consejo: Añadir blanco da tonos arcilla. Negro → tierra húmeda

🔸 Verde + Azul = Azul-verde

💚 Verde = Amarillo 60% + Azul 40%
🔵 Azul = 100% Azul
➡ Azul-verde frío, profundo
🔸Más Verde (70/30): verde azulado
Usos: musgo oscuro, hojas mojadas
🔸Más Azul (30/70): azul petróleo
Ideal para agua estancada, sombras frías
🌊Usos: Zonas húmedas, agua, sombra vegetal, líquenes
🎯Consejo: Agregar blanco da verde marino claro. Muy útil para ambientes húmedos

🔸 Verde + Amarillo = Amarillo-verde

💚 Verde = Amarillo 60% + Azul 40%
🟡 Amarillo = 100% Amarillo
➡ Verde vivo, brillante
🔸Más Verde (70/30): verde soleado
Perfecto para césped vivo, primavera
🔸Más Amarillo (30/70): verde limón
Ideal para musgo fresco, brotes
🌱Usos: Vegetación viva, campos, follaje joven
✅Consejo: Añadir blanco → verde pastel. Mezcla muy útil para luces en vegetación

🔸 Verde + Naranja = Tierra oscura

💚 Verde = Amarillo 60% + Azul 40%
🟧 Naranja = Rojo 50% + Amarillo 50%
➡ Marrón oscuro con matiz oliva
🔸Más Verde (70/30): marrón terroso verdoso
Perfecto para bases vegetales secas
🔸Más Naranja (30/70): marrón cálido
Ideal para madera envejecida, tierra húmeda
🌲Usos: Suelos de bosque, bases de dioramas naturales, madera sucia
🎯Consejo: Útil para hacer transiciones entre zonas verdes y marrones

🔸 Verde + Violeta = Gris verdoso

💚 Verde = Amarillo 60% + Azul 40%
🟣 Violeta = Rojo 50% + Azul 50%
➡ Gris apagado con matiz frío

Ideal para sombra de vegetación, camuflaje
🔸Más Violeta (30/70): gris violeta apagado
Perfecto para sombras de roca, ambientes oscuros
🌫Usos: Camuflaje, niebla vegetal, bases neutras
✅Consejo: Muy útil para zonas sombrías. Agregar blanco → gris militar claro

🔸 Verde + Amarillo-verde = Verde limón
