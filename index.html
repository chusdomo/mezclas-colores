🟡 AMARILLO con otros colores

Guía completa de mezclas y combinaciones cromáticas

🔸 Amarillo + Azul = Verde

🟡 Amarillo = 100%
🔵 Azul = 100%
➡ Verde puro = 50% + 50%
🔸Más Amarillo (70/30): verde amarillento
Perfecto para césped soleado, primavera
🔸Más Azul (30/70): verde frío
Ideal para hojas en sombra, algas, camuflaje
🌿Usos: Vegetación, moho, camuflaje, naturaleza
🎯Consejo: Añadir blanco = verde menta. Gris = verde oliva

🔸 Amarillo + Rojo = Naranja

🟡 Amarillo = 100%
🔴 Rojo = 100%
➡ Naranja saturado y cálido
🔸Más Amarillo (70/30): naranja claro
Usos: luces cálidas, zonas soleadas
🔸Más Rojo (30/70): naranja oscuro
Ideal para óxido, fuego, cueros
🔥Usos: Luces, fuego, objetos cálidos, ladrillos
⚠Consejo: Muy potente: suavizar con blanco o gris según uso

🔸 Amarillo + Violeta = Marrón sucio

🟡 Amarillo = 100%
🟣 Violeta = Rojo 50% + Azul 50%
➡ Marrón apagado, tierra neutra
🔸Más Amarillo (70/30): marrón cálido
Ideal para suelos secos, maderas claras
🔸Más Violeta (30/70): marrón oscuro violáceo
Usos: sombras de tierra, madera envejecida
🌰Usos: Tierra, madera, fondo vegetal oscuro
✅Consejo: Añadir blanco = beige sucio. Muy útil para bases

🔸 Amarillo + Verde = Amarillo-verde

🟡 Amarillo = 100%
💚 Verde = Amarillo 60% + Azul 40%
➡ Amarillo-verde, brillante y fresco
🔸Más Amarillo (70/30): verde limón
Perfecto para musgo fresco, primavera
🔸Más Verde (30/70): verde más oscuro
Usos: follaje, base de vegetación
🌱Usos: Césped, hojas tiernas, líquenes, algas
⚠Consejo: Es un color muy vivo: usar con control en zonas puntuales

🔸 Amarillo + Naranja = Amarillo-naranja

🟡 Amarillo = 100%
🟧 Naranja = Rojo 50% + Amarillo 50%
➡ Amarillo-naranja brillante
🔸Más Amarillo (70/30): amarillo dorado
Usos: focos, resplandores, piel cálida
🔸Más Naranja (30/70): naranja suave
Perfecto para piel, zonas con luz cálida
☀️Usos: Piel, telas, luz cálida, pintura de fuego
✅Consejo: Muy útil para tonos de carne si se suaviza con blanco

🔸 Amarillo + Azul-verde = Verde pastel

🟡 Amarillo = 100%
🌊 Azul-verde = Azul 60% + Verde 40%
➡ Verde claro, grisáceo, fresco
🔸Más Amarillo (70/30): verde limón pálido
Usos: vegetación joven, reflejos cálidos
🔸Más Azul-verde (30/70): verde grisáceo
Ideal para roca húmeda, musgo, óxido vegetal
🪨Usos: Vegetación seca, piedra, agua estancada
🎨Consejo: Muy útil como sombra cálida o luz vegetal desaturada

🔸 Amarillo + Rojo-violeta = Marrón rojizo

🟡 Amarillo = 100%
🍷 Rojo-violeta = Rojo 60% + Azul 40%
➡ Marrón rojizo cálido
🔸Más Amarillo (70/30): marrón claro tostado
Perfecto para tierra seca, bases de suelos
🔸Más Rojo-violeta (30/70): marrón vino
Ideal para madera barnizada, sombras calientes
🌾Usos: Tierra, madera, cuero viejo, telas nobles
🎯Consejo: Añadir blanco → beige vino. Buen tono de transición

🔸 Amarillo + Rojo-naranja = Naranja tostado

🟡 Amarillo = 100%
🟥 Rojo-naranja = Rojo 60% + Amarillo 40%
➡ Naranja cálido, como siena claro
🔸Más Amarillo (70/30): naranja ocre claro
Perfecto para zonas iluminadas, polvo, ropas viejas
🔸Más Rojo-naranja (30/70): naranja tostado
Usos: óxido, piedra cálida, piel curtida
🟫Usos: Ropas, bases de terreno seco, óxido moderado
✅Consejo: Muy versátil. Añadir gris o negro genera tonos tierra realistas

🔸 Amarillo + Azul-violeta = Gris oliva

🟡 Amarillo = 100%
🔵🟣 Azul-violeta = Azul 60% + Rojo 40%
➡ Gris con matiz verdoso, oliva apagado
🔸Más Amarillo (70/30): gris oliva claro
Ideal para rocas, bases de vegetación, camuflaje seco
🔸Más Azul-violeta (30/70): gris más frío
Perfecto para profundidad, sombras con tono verde
🎖Usos: Camuflaje, metales oxidados, desgaste vegetal
🎨Consejo: Añadir blanco = verde grisáceo muy realista

🔸 Amarillo + Amarillo-naranja = Mostaza

🟡 Amarillo = 100%
🍊 Amarillo-naranja = Amarillo 60% + Rojo 40%
➡ Mostaza cálido, rústico
🔸Más Amarillo (70/30): mostaza claro
Usos: polvo, piedra caliza, telas viejas
🔸Más Amarillo-naranja (30/70): ocre cálido
Ideal para ladrillos, óxido seco
🧱Usos: Madera vieja, óxido suave, texturas en roca o tierra
🎯Consejo: Muy útil en paisajes rurales o desérticos

🔸 Amarillo + Amarillo-verde = Verde seco

🟡 Amarillo = 100%
💚 Amarillo-verde = Amarillo 70% + Azul 30%
➡ Verde apagado, como liquen seco
🔸Más Amarillo (70/30): verde amarillento claro
Perfecto para musgo seco, heno
🔸Más Amarillo-verde (30/70): verde seco
Ideal para follaje sombreado, vegetación muerta
🌿Usos: Vegetación muerta, musgo, suelo
✅Consejo: Muy útil para dar realismo sin saturar la escena

🔸 Amarillo + Blanco = Crema

🟡 Amarillo = 100%
⚪ Blanco = 100%
➡ Crema, tono pastel cálido
🔸Más Amarillo (70/30): crema intenso
Ideal para efectos de sol, resplandores
🔸Más Blanco (30/70): crema pálido
Perfecto para telas, paredes, luz ambiente
🍦Usos: Luz cálida, piel clara, telas, comida
✅Consejo: Ideal como base clara para técnicas de iluminación

🔸 Amarillo + Negro = Ocre

🟡 Amarillo = 100%
⚫ Negro = 100%
➡ Ocre oscuro, marrón verdoso
🔸Más Amarillo (70/30): ocre cálido
Perfecto para tierra seca, rocas
🔸Más Negro (30/70): ocre frío
Usos: bases para sombra, metal envejecido
🏺Usos: Piedra, barro seco, óxido viejo
⚠Consejo: Se vuelve opaco rápidamente. Agrega blanco si oscurece demasiado

🔸 Amarillo + Gris = Mostaza grisáceo

🟡 Amarillo = 100%
⚪ Gris = Negro + Blanco
➡ Mostaza apagado, terroso
🔸Más Amarillo (70/30): mostaza cálido
Ideal para muros, desierto, polvo
🔸Más Gris (30/70): mostaza grisáceo
Usos: rocas, óxido muy viejo, ropa gastada
🏜Usos: Texturas terrosas, ropa, edificios envejecidos
🎨Consejo: Base ideal para técnicas de pincel seco
